# Node.js笔记

***

### Node与 ‘BOM’，’DOM‘两个没有关系

- js的三要素有ES，BOM，DOM，但Node与 ‘BOM’，’DOM‘两个没有关系

### cmd命令行

- 查看是否有安装node.js

  ```bash
  node -v
  ```

- 运行js文件

  ```bash
  node test.js
  ```

- 进入repl（交互式解析器，即可以在里面写js代码，没啥用）

  ```bash
  node
  ```

- 退出repl

  ```
  两次(Ctrl + C)
  ```

- 回到上一个目录

  ```bash
  cd ..
  ```

- 显示目录

  ```bash
  dir
  ```

- 清空上面的bash

  ```bash
  cls
  ```

- 删除文件

  ```bash
  del
  ```

### npm（node package message）

- 在node安装的时候已经自动安装好npm了
